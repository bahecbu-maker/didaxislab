📐 Desarrollo analítico

Con perímetro fijo P, si un lado mide x, el otro mide P/2 − x. El área es entonces una función cuadrática:

A(x) = x · (P/2 − x)

Esta función es una parábola invertida cuyo vértice indica el área máxima.

  1. Expandimos: A(x) = (P/2)·x − x²
  2. Derivamos e igualamos a cero: A'(x) = P/2 − 2x = 0 → x = P/4
  3. El otro lado también es P/2 − P/4 = P/4.

Conclusión: el máximo se alcanza cuando x = P/4, es decir, cuando el rectángulo es un cuadrado de lado P/4 y área (P/4)².
